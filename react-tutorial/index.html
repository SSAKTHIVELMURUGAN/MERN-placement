Difference Between Semantic and Non-Semantic Tags in HTML

Aspect Semantic Tags Non-Semantic Tags
Definition Tags that clearly describe their meaning and content. Tags that do not provide information about their content.
Examples
  • <header>
  • <nav>
  • <article>
  • <section>
  • <footer>
  • <div>
  • <span>
Usage Helps in understanding the structure and meaning of the content. Used for generic containers when semantic meaning is not needed.
Accessibility Improves accessibility for assistive technologies. Less helpful for accessibility purposes.
SEO Helps search engines better understand the content. Does not aid in SEO as much as semantic tags.

Difference Between Internal CSS and External CSS

Aspect Internal CSS External CSS
Definition CSS styles defined within the <style> tag in the HTML document. CSS styles defined in a separate .css file linked to the HTML document.
Location Inside the <head> section of the HTML document. In a separate file, typically with a .css extension.
File Management Styles are embedded directly in the HTML file. Styles are placed in an external file, separate from the HTML.
Use Case Suitable for small projects or single-page documents. Suitable for larger projects or sites with multiple pages.

Attributes in HTML

  • id Attribute: Uniquely identifies a single HTML element.
  • class Attribute: Assigns one or more classes to an element, allowing for shared styling.
  • name Attribute: Used to reference form elements during data submission.
  • data Attributes: Used to store custom data private to the page or application.
  • Global Attributes: Attributes that can be used on any HTML element, such as style, title, lang, dir, hidden, and tabindex.

CSS Flexbox Overview

CSS Flex Container

display
Defines the element as a flex container, enabling flexbox layout for its children (flex items). Values include flex and inline-flex.
flex-direction
Specifies the direction of the main axis along which flex items are laid out. Values include row, row-reverse, column, and column-reverse.
flex-wrap
Controls whether flex items are forced onto a single line or can wrap onto multiple lines. Values include nowrap, wrap, and wrap-reverse.
flex-flow
Shorthand for setting both flex-direction and flex-wrap in one declaration. Syntax: flex-flow: <flex-direction> <flex-wrap>.
justify-content
Aligns flex items along the main axis. Values include flex-start, flex-end, center, space-between, space-around, and space-evenly.
align-items
Aligns flex items along the cross axis (perpendicular to the main axis). Values include flex-start, flex-end, center, baseline, and stretch.
align-content
Aligns lines of flex items when there is extra space in the cross axis (multiline flex containers). Values include flex-start, flex-end, center, space-between, space-around, and stretch.

CSS Flex Items

flex
Shorthand for setting flex-grow, flex-shrink, and flex-basis of a flex item. Syntax: flex: <flex-grow> <flex-shrink> <flex-basis>.
flex-grow
Specifies how much a flex item should grow relative to the other flex items. Default is 0.
flex-shrink
Specifies how much a flex item should shrink relative to the other flex items. Default is 1.
flex-basis
Defines the initial size of a flex item before additional space is distributed. Default is auto.
order
Controls the order of flex items within a flex container. Default is 0.
align-self
Allows a flex item to override the align-items value for its own alignment. Values include auto, flex-start, flex-end, center, baseline, and stretch.

CSS Flex Responsive

Media Queries
Used to apply different styles for different screen sizes. Example syntax: 
@media (max-width: 768px) {
    /* Styles applied when screen width is 768px or less */
}
Example: Responsive Flex Container
.flex-container {
    display: flex;
    flex-wrap: wrap;
    justify-content: center;
}

@media (max-width: 768px) {
    .flex-container {
        flex-direction: column;
        align-items: center;
    }
}
Example: Responsive Flex Item
.flex-item {
    flex: 1 1 200px;
}

@media (max-width: 768px) {
    .flex-item {
        flex: 1 1 100%;
    }
}

Summary

CSS Flexbox provides a powerful layout mechanism for creating flexible and responsive designs. Understanding these properties allows you to effectively structure and align content in modern web layouts, adapting to various screen sizes and orientations. Adjust these properties based on your specific layout requirements to achieve optimal design flexibility and responsiveness.
